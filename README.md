## 📜 Genesis & Evolution

**Hypruccin** — the very portmanteau itself bears witness to its origin: a humble devotion to the **Catppuccin** palette, whose mauve-toned reveries first kindled this undertaking. Yet a configuration, once set into motion, refuses to remain merely decorative; it accumulates purpose, acquires character, and evolves into something greater than the sum of its chroma.

What began as a simple act of homage soon transcended its chromatic confines. This repository now embodies a more ambitious aspiration — to construct adesktop environment that resonates with intentionality: every keybinding deliberate, every daemon purposeful, every visual element a considered choice. The palette remains, honored still, yet it now serves as merely the foundation upon which a more expansive philosophy is erected.

**Hypruccin** is my personal Arch Linux configuration built around **Hyprland** and **UWSM**.  
Unlike the Matugen edition that embraces dynamic color generation from wallpapers, this version adopts a curated, hand-picked aesthetic philosophy inspired by the **Omarchy** design principles — where every element serves a purpose, and beauty emerges from thoughtful restraint.

This setup reflects my journey toward a desktop environment that feels cohesive, minimal, and deeply personal. I deliberately excluded applications that didn't align with this vision, focusing instead on tools that enhance productivity without compromising elegance.

### 🎯 Design Philosophy

- **Intentional Minimalism:** Every program has a reason to exist. No bloat, no clutter.
- **Omarchy Inspiration:** Clean lines, purposeful typography, and a coherent visual language.
- **Static Elegance:** Unlike dynamic approaches, this configuration embraces carefully selected themes that remain consistent and reliable.

> 💡 **Curious about dynamic theming?** I also maintain a **[Matugen Edition](https://github.com/imanubdesigner/hypruccin/tree/hypruccin-matugen)** branch that generates colors directly from your wallpaper — for those who prefer a living, breathing aesthetic.

## 🛠️ Utilities & Enhancements

- 🐱 [**Bat**](https://github.com/sharkdp/bat) + [**Bat-extras**](https://github.com/eth-p/bat-extras): Replacement for `cat` with syntax highlighting.
- 📊 [**Btop**](https://github.com/aristocratos/btop): Visual resource monitor in terminal.
- 🎵 [**Cava**](https://github.com/karlstav/cava): Real-time audio visualizer.
- 🐘 [**Elephant**](https://github.com/abenz1267/elephant): Backend search and indexing engine for Walker.
- ⚡ [**Fastfetch**](https://github.com/fastfetch-cli/fastfetch): System info fetch tool.
- 🌐 [**Fcitx5**](https://github.com/fcitx/fcitx5): Input method framework for multilingual and emoji support.
- 🔍 [**Fzf**](https://github.com/junegunn/fzf): Blazing-fast fuzzy finder used in scripts.
- 🌄 [**Awww**](https://github.com/LGFae/swww): Wallpaper daemon with smooth transitions.
- 🔊 [**WirePlumber**](https://pipewire.pages.freedesktop.org/wireplumber/): Session manager for PipeWire.
